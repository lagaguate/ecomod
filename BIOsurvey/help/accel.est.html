accel.est {BIOsurvey}R Documentation

Calculation of acceleration estimate

Description

Calculation of acceleration estimate for the BCa bootstrap confidence intervals as per Efron and Tibshirani (1993).

Usage

accel.est(data)

Arguments

data List containing observations in each stratum (as a list object, one element per stratum), number of observations per stratum and Stratum labels

Details

This is an internal function called by accel.str and is not intended for stand-alone use.

Author(s)

Stephen J. Smith, smithsj@mar.dfo-mpo.gc.ca

References

Efron, B., and Tibshirani, R.J. (1993). An introduction to the bootstrap. Chapman & Hall, London.

See Also

accel.str, summary.boot
