summary.boot {BIOsurvey}R Documentation

Summary of boot strata object

Description

This function is a method for the generic function summary for class boot.

Usage

## S3 method for class 'boot':
summary(object, CI.method = c("Percentile", "BC", "BCa"), alpha.b = 0.05, ...)

Arguments

object boot object created by boot.strata.
CI.method choose one of three confidence interval methods (Efron and Tibshirani, 1993) for the bootstrap estimate of the stratifed mean. Percentile = empirical percentiles; BC = bias-corrected; BCa = bias corrected and adjusted. The BCa method requires the acceleration estimation provided in boot.strata. This confidence interval method will produce an error statement if accel=NA.
alpha.b alpha level for confidence interval.
... further arguments passed to or from other methods.

Details

This function is a method for the generic function summary for class boot.

Value

This function summarizes a boot object by printing out the original stratified mean, variance of stratified mean, bootstrap stratified mean and the variance of the bootstrap estimate. In addition to the upper and lower confidence limits, the length and shape of the confidence interval are also given. The shape is calculated as the natural log of the ratio of the upper limit minus the median to the median minus the lower limit (Efron, 1992). A summary of the bootstrap estimates is also provided using quantile.

References

Efron, B. (1992). Jackknife-after-bootstrap standard errors and influence functions (with discussion). J. Royal Stat. Soc. Ser. B 54: 83–127.

Efron, B. and R.J. Tibshirani. (1993). An introduction to the bootstrap. Chapman & Hall, London.

Smith, S.J. (1997). Bootstrap confidence limits for groundfish trawl survey estimates of mean abundance. Can. J. Fish. Aquat. Sci. 54: 616–630.
