Association.plot {BIOsurvey}R Documentation

Plots association between catch and covariate

Description

Produces an association plot using one environmental variable for stratified random survey data.

Usage

Association.plot(data, hydro, strata.group, species = NULL,subset, plot = TRUE, ...)

Arguments

data data.frame (class = strata.data) prepared by Prepare.strata.data
hydro character name of vector specifying hydrographic variable to be analysed
strata.group List object prepared by Prepare.strata.file
species character name of element in data.obj specifying what species' catches are to be analysed. The default is the hydrographic variable named in hydro= argument.
subset expression specifying which subset of observations should be used in the fit. This can be a logical vector (which is replicated to have length equal to the number of observations), a numeric vector indicating the observation numbers to be included, or a character vector of the observation names that should be included. All observations are included by default.
plot logical flag: if TRUE, then quantile-quantile plot is plotted on current graphics device. Otherwise x and y coordinates for plot provided.
... arguments to be passed onto plot function, such as xlab, xlim, etc.

Details

See example below for producing an association plot comparing fish catch and a hydrographic variable.

Value

if plot is FALSE, quantile.strata returns a list with components x and y, giving coordinates of the points that would have been plotted.

References

Smith, S.J. (1996). Analysis of data from bottom trawl surveys. In Assessment of groundfish stocks based on bottom trawl survey results. Lassen, H. (ed.). NAFO Scientific Council Studies. 28: 25–53.

See Also

Prepare.strata.data, Prepare.strata.file, Stratify, Association.test

Examples

#plot cumulative frequency curve for Depth.



Association.plot(Scallop.stratadata, hydro="DEPTH", strata.group=Scallop.strata,
xlab="Depth (m)")

#add line for Commercial size scallops and Depth.

lines(Association.plot(Scallop.stratadata, hydro="DEPTH", strata.group=Scallop.strata,
species="Commercial",plot=FALSE),lty=2)
