influential.strata {BIOsurvey}R Documentation

Diagnostic plot for influential observations

Description

Produces a plot showing the proportion that each catch (on abscissa) contributes to the stratified mean (on ordinate).

Usage

influential.strata(object, Plot = TRUE, ...)

Arguments

object strata object created by Stratify.
Plot logical flag: if TRUE, then influential plot is plotted on current graphics device. Otherwise x and y coordinates for plot provided.
... arguments to be passed onto plot function, such as xlab, xlim, etc. The labels and limits for the y-axis are already set.

Details

This function uses the generic function plot to produce this special plot for class strata.

Value

if plot is FALSE, influential.strata returns a list with components x and y, giving coordinates of the points that would have been plotted. Strata labels are also included.

References

Smith, S.J. (1996). Analysis of bottom trawl survey data, In Assessment of groundfish stocks based on bottom trawl survey results. H. Lassen (Ed.) NAFO Scientific Council Studies 28: 25–53.

See Also

Stratify,Identify.influential.plot

Examples




influential.strata(Scallop.numbers.2008,Plot=TRUE,xlab="Scallop numbers")

Fluke.influential<-influential.strata(Scallop.numbers.2008,Plot=FALSE)
