accel.str {BIOsurvey}R Documentation

Calculate stratified acceleration estimate

Description

Calculate stratified acceleration estimate for BCa confidence interval as per Smith (1997).

Usage

accel.str(x.str)

Arguments

x.str List containing observations in each stratum (as a list object, one element per stratum), number of observations per stratum and Stratum labels

Details

This is an internal function called by summary.boot and is not intended for stand-alone use.

Author(s)

Stephen J. Smith, smithsj@mar.dfo-mpo.gc.ca

References

Smith, S.J. (1997). Bootstrap confidence limits for groundfish trawl survey estimates of mean abundance. Can. J. Fish. Aquat. Sci. 54: 616–630.

See Also

summary.boot, accel.est
