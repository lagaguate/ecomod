Stratify {BIOsurvey}R Documentation

Calculate stratified estimates

Description

Selects data and calculates within strata estimates of mean and variance.

Usage

Stratify(data.obj, strata.group, species, subset)

Arguments

data.obj data.frame (class = strata.data) prepared by Prepare.strata.data.
strata.group List object prepared by Prepare.strata.file.
species name of element in data.obj specifying what species' catches are to be analysed.
subset expression specifying which subset of observations should be used in the fit. This can be a logical vector (which is replicated to have length equal to the number of observations), a numeric vector indicating the observation numbers to be included, or a character vector of the observation names that should be included. All observations are included by default.

Details

The data object can contain survey data for more than just the strata of interest. Selection of which strata are to be included are specified with strata.group.

Missing values (NA's) are allowed for all arguments and are automatically removed from the analysis.

Value

stratify returns a list of class strata with required components for further analyses of the data. If the result is unassigned, print.strata will automatically give means, variances, sample sizes and the stratum weight for each stratum.

yhi the number of animals caught in tow i in strata h.
Strata the stratum label for each tow.
Nh the total number of population sample units in stratum h.
Wh the proportion of the area in stratum h.
nh the total number of sampled units in stratum h.
descrip the total number of sampled units in stratum h.

References

Cochran, W.G. (1977). Sampling Techniques. John Wiley & Sons. New York, N.Y.

Smith, S.J. (1996). Analysis of bottom trawl survey data, In Assessment of groundfish stocks based on bottom trawl survey results. H. Lassen (Ed.) NAFO Scientific Council Studies 28: 25–53.

See Also

Prepare.strata.data, print.strata, summary.strata.

Examples



Scallop.numbers.2008<-Stratify(Scallop.stratadata,Scallop.strata,
species=Commercial)
