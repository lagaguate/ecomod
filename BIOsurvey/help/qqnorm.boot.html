qqnorm.boot {BIOsurvey}R Documentation

Quantile-quantile plot for bootstrap data

Description

Produce a quantile-quantile plot for bootstrap data from boot objects.

Usage

## S3 method for class 'boot':
qqnorm(y, ylab = "", ...)

Arguments

y boot object created by boot.strata
ylab text for ylab of plot
... further arguments passed to or from other methods.

Details

Quantile-quantile plot of bootstrap estimates is produced on current graphics device.

References

Smith, S.J. (1996). Analysis of data from bottom trawl surveys. In Assessment of groundfish stocks based on bottom trawl survey results. Lassen, H. (ed.). NAFO Scientific Council Studies. 28: 25–53.

See Also

boot.strata

Examples

## Not run: 
qqnorm.boot(Scallop.numbers.2008.boot,ylab="Stratified mean")
## End(Not run)
