Prepare.strata.file {BIOsurvey}R Documentation

Prepare strata information for stratified analysis

Description

This can be a user written function which produces a list object containing, the strata numbers, area of each stratum and total number of samples in each stratum to be used for stratum weights.

Usage

Prepare.strata.file(data)

Arguments

data data object containing at a minimum strata numbers or labels and the total number of samples in a strata. In the scallop data supplied with this package, the latter information is contained in the column "NH". If the user's file contains other heading names, then this function should be rewritten to provide the list object given below.

Details

Other information such as comments on method used to calculate total number of samples, etc., can also be added to the list object. The stratified analysis will be conducted for all of the strata included in this list object. Modifications to this list object can be used to select subsets of strata to be included in analysis.

Value

a list object with at least the following components

Strata Strata numbers or identifiers.
NH Total number of samples possible in each stratum
