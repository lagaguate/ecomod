Prepare.strata.data {BIOsurvey}R Documentation

Prepares data for the analysis tools in the NMFSsurvey package

Description

This function expects data in the standard form from the export option of the NEFSC package SAGE.

Usage

Prepare.strata.data(data)

Arguments

data is the data.frame obtained from read.table("file.name",head=TRUE), where file.name is the ascii file exported from the NEFSC survey analysis system SAGE.

Details

This function can be replaced by any other function which assigns the classes below and sets a column containing strata identifiers called Strata. Also replaces all underscores in variable names with periods.

Value

produces a data frame of class "data.frame" and "strata.data".

Author(s)

Stephen J. Smith, smithsj@mar.dfo-mpo.gc.ca

Examples

## Not run: 
Scallopsurveydata<-read.table("Scallopsurveydata.txt",head=TRUE)

Scallop.stratadata<-Prepare.strata.data(Scallopsurveydata)
## End(Not run)
