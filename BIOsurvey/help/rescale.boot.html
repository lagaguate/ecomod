rescale.boot {BIOsurvey}R Documentation

Calculate rescale bootstrap

Description

Calculates rescale bootstrap as per Rao and Wu (1988).

Usage

rescale.boot(X, nred)

Arguments

X vector of observations
nred number to reduce the sample size by for rescale bootstrap (1 or 3)

Details

This is an internal function called by boot.strata and is not meant to be used on it's own.

Author(s)

Stephen J. Smith, smithsj@mar.dfo-mpo.gc.ca

References

Rao, J.N.K. and C.F.J. Wu. (1988). Resampling inference with complex survey data. J. Am. Stat. Assoc. 83: 231–241.
