Print method for boot objects

Description

This function provides a basic summary of a bootstrap object for stratified means.

Usage

## S3 method for class 'boot':
print(x, ...)

Arguments

x boot object created by boot.strata.
... further arguments passed to or from other methods.

Details

this function is a method for the generic function print for class boot.

Value

orig.mean original stratified mean.
orig.var original variance of stratified mean.
Number of bootstraps number of replications.
Bootstrap mean mean of bootstrap stratified means.
Average Bootstrap Variance mean variance calculated from each bootstrap resample.
