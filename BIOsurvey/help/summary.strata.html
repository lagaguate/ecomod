summary.strata {BIOsurvey}R Documentation

Summary method for strata objects

Description

Summary of strata object including stratified mean and total, respective standard error and conventional confidence intervals.

Usage

## S3 method for class 'strata':
summary(object, alpha.t = 0.05, effic = FALSE, nopt = FALSE, ...)

Arguments

object strata object created by Stratify
alpha.t the alpha level for the student's-t confidence intervals. Default is 0.05. Confidence intervals also provided.
effic logical flag: if TRUE, then design-efficiency estimates for the allocation and stratification components are calculated. Estimated maximum efficiency for optimal allocation for current stratification also calculated.
nopt logical flag: if TRUE, then optimal sample sizes per stratum are calculated based on current estimates of strata variances. Compromise allocation for case where there must be a minimum of 2 samples per strata also calculated. This option can only be calculated if sample size greater than 1 in all strata.
... further arguments passed to or from other methods.

Details

Student's-t confidence intervals calculated using Satterthwaite's approximation (Cochran 1977). Efficiency estimates based on Sukhatme and Sukhatme's (1953) approximation (Smith and Gavaris 1993). This function is a method for the generic function summary for class strata.

Value

A data.frame containing the following:

yst estimate of the stratified mean.
se.yst estimate of standard error of mean.
Yst estimate of stratified total.
df.yst effective degrees of freedom.
alpha alpha level for confidence interval.
ci.yst lower and upper limits for confidence interval.
effic.alloc allocation component of estimated efficiency as a percentage of simple random sampling variance.
effic.str strata component of estimated efficiency as a percentage of simple random sampling variance.
var.ran estimated variance of the mean for a simple random sample.
max.eff estimated maximum efficiency for optimal allocation for current stratification.
nopt A matrix of the optimal and compromise sample sizes (minimum sample size = 2) for each stratum.
descrip Simple label ``Stratified Analysis'' for summary object.

References

Cochran, W.G. (1977). Sampling Techniques. John Wiley & Sons. New York, N.Y.

Smith, S.J. and S. Gavaris. (1993). Improving the precision of fish abundance estimates of Eastern Scotian Shelf cod from bottom trawl surveys. North American Journal of Fisheries Management. 13: 35–47.

Sukhatme, R.V., and B.V. Sukhatme. (1970). Sampling theory of surveys with applications. Iowa State University Press, Ames, Iowa.
