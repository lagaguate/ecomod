Identify.influential.plot {BIOsurvey}R Documentation

Indentifies influential observations

Description

Identifies observations that contribute disproportionately to the stratified mean.

Usage

Identify.influential.plot(x,...)

Arguments

x object produced by influential.strata
... further arguments passed to par such as cex, col and font.

Details

This function uses indentify to label points using the curser. Point labels refers to Strata where observation is located.

Value

Points labelled on plot. An integer vector containing the indexes of the identified points.

See Also

influential.strata, identify

Examples
