Association.test {BIOsurvey}R Documentation

Test association between catch and covariate

Description

This function produces a randomization test incorporating strata weights for associating survey catches with coincidental observations on environmental variables.

Usage

Association.test(data, hydro, strata.group, species, nreps = 0, 
method = c("KS-Test", "CVM-Test")[1],subset)

Arguments

data data.frame (class = strata.data) prepared by Prepare.strata.data
hydro character name of vector specifying hydrographic variable to be analysed
strata.group List object prepared by Prepare.strata.file
species character name of element in data.obj specifying what species' catches are to be analysed. The default is the hydrographic variable named in hydro= argument.
nreps number of replications used to form null hypothesis distribution.
method test method to be used. Choices are modifications of Kolmogorov-Smirnov type test or Cramer von-Mises type test. The former test is the default.
subset expression specifying which subset of observations should be used in the fit. This can be a logical vector (which is replicated to have length equal to the number of observations), a numeric vector indicating the observation numbers to be included, or a character vector of the observation names that should be included. All observations are included by default.

Details

WARNING! Although, the randomization algorithms has been written as efficiently as possible they are still written in R code and may take a few minutes. An implementation using C code with a dynamic load will be considered for a future version.

Value

an object of class assoc is produced which contains the following items.

test.stat test statistic for method chosen.
test.stat.rep vector containing a test statistic value for each replication of the randomization test. These values comprise the null hypothesis distribution.
p.level p level for the test statistic compared to the distribution in test.stat.rep.
prefer.lab label containing the type of test chosen.
prefer2.lab label for summary.assoc.
prefer3.lab label for summary.assoc.
descrip label indicating that this object contains the result of an ``Association Analysis''.

References

Perry, R.I. and S.J. Smith. (1994). Identifying habitat associations of marine fishes using survey data: an application to the Northwest Atlantic. Can. J. Fish. Aquat. Sci. 51: 589–602.

Smith, S.J. (1996). Analysis of data from bottom trawl surveys. In Assessment of groundfish stocks based on bottom trawl survey results. Lassen, H. (ed.). NAFO Scientific Council Studies. 28: 25–53.

See Also

Prepare.strata.data, Prepare.strata.file, Stratify, summary.assoc

Examples




Scallop.numbers.2008.assoc<-Association.test(Scallop.stratadata, 
hydro="DEPTH", strata.group=Scallop.strata, species="Commercial", nreps=1000, 
method="CVM-Test")
