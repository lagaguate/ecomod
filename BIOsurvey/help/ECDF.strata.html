ECDF.strata {BIOsurvey}R Documentation

Plot empirical cumulative distribution plot

Description

Constructs a empirical cumulative distribution plot for a strata object

Usage

ECDF.strata(object, Plot = TRUE, ...)

Arguments

object strata object created by Stratify.
Plot logical flag: if TRUE, then cumulative distribution is plotted on current graphics device. Otherwise x and y coordinates for plot provided.
... arguments to be passed onto plot function, such as xlab, xlim, etc. The labels and limits for the y-axis are already set.

Details

This function uses the generic function plot to produce this special plot for class strata. The empirical cumulative distribution function is derived using the ratio of the stratum weight and sample size as the weight for each observation.

Value

if plot is FALSE, quantile.strata returns a list with components x and y, giving coordinates of the points that would have been plotted.

References

Chambers, R. L. and R. Dunstan. (1986). Estimating distribution functions from survey data. Biometrika. 73: 597–604.

See Also

Stratify

Examples




ECDF.strata(Scallop.numbers.2008,Plot=TRUE,xlab="Scallop numbers")
